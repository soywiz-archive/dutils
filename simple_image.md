simple\_image is a very lightweight library for load, manipulate and save color-mapped and true color images without any external dependence.

Reading:
  * BMP
  * TGA
  * PNG
  * GIM
  * TM2
  * PSD

Writing:
  * TGA
  * PNG
  * GIM _(updating)_
  * TM2 _(updating)_

Now it supports font drawing using FreeType.
